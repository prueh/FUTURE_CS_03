# Cyber Security Awareness Presentation (Task 03)

## Project Overview
This project was developed for the *Future Interns Cybersecurity Internship* to promote security best practices within an organization. The goal was to create a high-impact, visual awareness guide that helps non-technical staff identify and mitigate common digital threats.

## Key Awareness Topics
* *Phishing Identification:* Analyzing sender domains, spotting "artificial urgency," and performing link hover checks.
* *Multi-Factor Authentication (MFA):* Explaining why MFA blocks 99% of automated attacks and encouraging the use of Authenticator Apps.
* *Digital Hygiene:* Best practices for creating strong passphrases, keeping systems updated, and securing physical workstations (Win + L).
* *Incident Reporting:* A clear 3-step protocol for employees who suspect they have been compromised.

## Technical Implementation
* *HTML5/CSS3:* Structured for a 1280x720 "Slide" experience.
* *Responsive Design:* Uses a professional White & Blue theme for high readability and executive appeal.
* *Font Awesome:* Integrated iconography for intuitive navigation and visual storytelling.
